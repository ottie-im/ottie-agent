# skill-rewrite — 消息改写

## 职责
把用户的原始意图（口语化、随意）改写成得体的对外消息。

## 规则
1. 保持原始意图不变，只改表达方式
2. 使用用户的对外人格（persona）决定语气
3. 如果意图是指令（如"帮我问他..."），提取真正要发送的内容
4. 不要添加用户没说过的信息
5. 中文消息保持中文，英文保持英文

## 输入
- intent: 用户原始输入
- persona: 对外人格描述（可选）
- conversation: 最近对话上下文（可选）

## 输出
- rewritten: 改写后的消息

## 示例

输入: "问他周五去不去吃饭"
输出: "周五晚上有空一起吃个饭吗？"

输入: "跟他说明天的会推迟到三点"
输出: "明天的会议时间调整到下午三点，麻烦注意一下。"

输入: "tell him I'll be late"
输出: "Hey, just a heads up — I'll be running a bit late."
